Recap

Recap: Lexical Decision

Participants decide whether a letter string is a word or nonword

Lower RT means faster recognition

Common words are usually recognised more quickly

Similar words can create lexical competition

Dataset

Meet the dataset

A prepared teaching version of a public lexical decision megastudy

Base source: English Lexicon Project

Each row is one English word

This is an item analysis of word-level means

Variables

What the core variables mean

word

the lexical item

length

number of letters

log_frequency

how common the word is (higher = more frequent)

AoA

estimated age at which the word is learned (higher = learned later)

OLD20

a measure of orthographic neighbourhood similarity (lower = more similar neighbours)

mean_RT_ms

average lexical decision response time

Key predictor

AoA

AoA stands for Age of Acquisition

Estimated age when a word is learned

Lower AoA = learned earlier; higher AoA = learned later

Earlier-acquired words are often recognised faster (e.g. Morrison & Ellis, 2000)

AoA can matter even after controlling for frequency

Key predictor

OLD20

OLD20 is a measure of how similar a word is to other words in spelling

Based on the word’s 20 closest spelling neighbours

Lower OLD20 = more similar neighbours; higher OLD20 = more distinct

Similar-looking words may increase lexical competition

More graded than a simple neighbour count

Regression

Regression recap 1: what regression does

Regression tests whether one variable predicts another

It links a dependent variable to one or more predictors

It tells us about direction, size, and uncertainty

It moves us beyond simple description

Why multiple regression

Regression recap 2: why multiple regression helps

Multiple regression includes several predictors in the same model

It shows whether each predictor explains unique variation

Coefficients are interpreted holding other predictors constant

This matters when predictors overlap

Interpretation

Turning coefficients into claims

For each predictor, note:

  • coefficient sign
    • confidence interval or p value
    • plain-language interpretation

Ask:

Better than “Which is strongest?”:

  • Which predictors show the clearest unique association?
