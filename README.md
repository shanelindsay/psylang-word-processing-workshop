# Word Processing Workshop Files

Student download pack for the Psychology of Language word-processing workshop.

## Current files

- `data/word-processing-teaching-dataset.csv`
  - Main teaching dataset for the workshop
  - Core columns include `word`, `mean_RT_ms`, `length`, `log_frequency`, `AoA`, and `OLD20`
  - Additional columns such as `mean_accuracy`, `concreteness`, and `contextual_diversity` are also present

- `data/word-processing-trial-level-data.csv`
  - Trial-level lexical decision dataset, with one row per trial rather than one row per word
  - Includes response-level fields such as `Sub_ID`, `Trial`, `Type`, `D_RT`, `D_Word`, `Outlier`, and `D_Zscore`
  - Useful for showing where the summary teaching dataset comes from and for item- or participant-level extensions

- `data/data_dictionary.md`
  - Short glossary for the main workshop variables

- `handout/L6_student_guide.docx`
  - Current student handout for the workshop

- `slides/L6_word_processing_workshop_slides.pdf`
  - Workshop slide deck in PDF format

- `slides/L6_word_processing_workshop_frontend.html`
  - HTML export of the workshop slides
