# Word Processing Workshop Files

Student download pack for the Psychology of Language word-processing workshop.

## Included files

- `data/word-processing-teaching-dataset.csv`
  - Current teaching dataset for the workshop
  - Based on `workshop/data/l6-word-processing/processed/elp_lexical_decision_teaching.csv`
  - Core columns include `word`, `mean_RT_ms`, `length`, `log_frequency`, `AoA`, and `OLD20`
  - Additional columns such as `mean_accuracy`, `concreteness`, and `contextual_diversity` are also present

- `data/word-processing-trial-level-data.csv`
  - Trial-level lexical decision data
  - Based on `workshop/data/LexicalData.csv`
  - Includes fields such as `Sub_ID`, `Trial`, `Type`, `D_RT`, `D_Word`, `Outlier`, and `D_Zscore`
  - Optional extension file rather than the main live workshop dataset

- `handout/word-processing-student-guide-DRAFT.md`

- `slides/word-processing-workshop-slides.pdf`

- `reference/data_dictionary.md`
  - Short glossary for the main workshop variables
